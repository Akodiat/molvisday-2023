# Simulation of a DNA origami linear actuator

Slider and rail designed in caDNAno and converted assembled, and prepared for simulation in oxView (oxview.org)
Simulated for 1e9 timesteps using the oxDNA2 model (Simulated by Erik Benson).

rail.json
:  caDNANO Design file for the dna origami rail

slider.json
:  caDNANO Design file for the dna origami slider

linear_actuator.top
:  oxDNA topology file for slider-rail assembly

linear_actuator.dat
:  oxDNA configuration file for slider-rail assembly (initial configuration)

aligned_traj.dat
:  Aliged oxDNA trajectory file for slider-rail assembly

Slider design source:

Benson, E., Carrascosa Marzo, R., Bath, J., & Turberfield, A. J. (2021). Strategies for constructing and operating DNA origami linear actuators. Small, 17(20), 2007704.
https://onlinelibrary.wiley.com/doi/10.1002/smll.202007704
